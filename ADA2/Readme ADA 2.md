Desarrollamos un simulador de atención al cliente para un banco utilizando la estructura de datos Cola (Queue). El programa modeló el sistema con tres cajas de atención (tres colas), donde los clientes se generaban con tiempos de transacción aleatorios y se asignaban al azar a una de ellas. La simulación rastreó el ingreso y la salida de los clientes, y calculó métricas clave como el tiempo promedio de espera. Esta actividad reforzó el entendimiento del principio FIFO (First-In, First-Out) y demostró cómo las colas son fundamentales para modelar procesos de orden de llegada en sistemas reales y operativos.
