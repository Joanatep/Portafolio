En esta actividad implementamos un sistema de gestión de estudiantes utilizando la estructura de datos **Árbol Binario de Búsqueda** (**ABB**). La **matrícula** del estudiante sirvió como clave numérica para ordenar la estructura, lo que permitió la implementación eficiente de operaciones fundamentales como la inserción, eliminación y actualización de datos. Además de estas operaciones, el sistema generó reportes complejos y ordenados. Este ejercicio fue clave para dominar el concepto de ABB, comprobar su velocidad para operaciones basadas en la clave de ordenamiento, y aplicar la **recursividad** en el manejo de estructuras dinámicas complejas.
